# Forge v0 Product Scope

Forge v0 is a local-first Streamlit app for mapping dry-lab biomedical research workflows.

## Goal

Help a lab understand what datasets, scripts, notebooks, outputs, figures, models, and reports exist in a project, what created what, what is current, and what is approved.

## In Scope

- View one project
- Add/edit artifacts
- Link artifacts with relationships
- View artifact table
- View pipeline graph
- Mark status/current
- Create a release
- Show activity history

## Out of Scope

- Cloud
- Login/auth
- AI agents
- Google Drive sync
- GitHub sync
- File uploads
- Real version control
- HIPAA/security system
- Multi-user collaboration
- Billing
